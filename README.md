# MEU PORTFOLIO

## Como executá-lo?

### Instalar todas as dependências
npm install

git merge branch_name - mescla branch
git branch branch_name - cria branch
git branch - diz as branchs que tem
git branch -D branch_name - deleta branch

### Executar
